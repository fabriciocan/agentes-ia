## Integração WhatsApp Cloud API

A maior atualização da plataforma até agora. Adicionamos suporte nativo à **WhatsApp Cloud API** (API oficial do Meta), que oferece maior confiabilidade, limites de mensagem superiores e conformidade com os termos de serviço do WhatsApp.

### Novidades

**Meta Embedded Signup**
- Fluxo guiado para conectar a conta do WhatsApp Business sem sair da plataforma
- Aprovação automática de números via processo oficial do Meta
- Gestão de permissões e tokens diretamente no painel admin

**Suporte a Mensagens Interativas**
- Botões de resposta rápida nas mensagens do agente
- Listas de opções para fluxos estruturados
- Templates aprovados pelo Meta para mensagens proativas

**Webhook Bidirecional**
- Recebimento de mensagens via webhook seguro com verificação de assinatura
- Envio de respostas com retry automático em caso de falha
- Logs detalhados de entrega e leitura

### Melhorias de Infraestrutura

- Filas de mensagens com Redis para processamento assíncrono de alto volume
- Rate limiting inteligente respeitando os limites da API do Meta
- Monitoramento de saúde da conexão com alertas automáticos

### Migração

Clientes usando Evolution API podem migrar gradualmente. Ambas as integrações continuam suportadas e podem coexistir na mesma conta.
