## Base de Conhecimento Inteligente

Lançamos o sistema de base de conhecimento com RAG — a tecnologia que elimina alucinações e faz o agente responder com base nos **seus** documentos.

### Novidades

**Upload de Documentos**
- Suporte a PDF, DOCX, XLSX, TXT e CSV
- Upload via interface drag-and-drop ou URL
- Processamento assíncrono com barra de progresso em tempo real
- Limite de 500 MB por base (plano Professional) e ilimitado (Enterprise)

**Motor de Busca Semântica (Qdrant)**
- Banco de dados vetorial de alta performance rodando localmente
- Embeddings com OpenAI `text-embedding-3-small` para máxima precisão
- Chunking inteligente com sobreposição de 20% para preservar contexto
- Busca por similaridade com threshold configurável

**Pipeline RAG com LangChain**
- Recuperação dos top-K chunks mais relevantes por query
- Injeção automática no contexto do agente
- Histórico de conversa preservado para contexto multi-turno
- Keywords automáticas por chunk para busca híbrida

**Gerenciador de Base de Conhecimento**
- Interface para visualizar, editar e excluir documentos
- Preview dos chunks indexados
- Teste de queries direto no painel
- Métricas de cobertura e qualidade da base

### Melhorias nos Agentes

- Configuração de instrução de sistema (system prompt) por agente
- Temperatura configurável do modelo (criatividade vs precisão)
- Definição de fallback quando o agente não sabe a resposta
- Ativação/desativação da base de conhecimento por agente

### Correções

- Respostas duplicadas em conversas de alta frequência
- Timeout em uploads de documentos grandes (>50MB)
- Ordenação incorreta de mensagens no histórico de conversas
